## How to Add Logos:
1. Download SVG logos from [Simple Icons](https://simpleicons.org/) or [DevIcon](https://devicon.dev/)
2. Place them in the `public/logos/` directory
3. The app will use placeholders if logos are not found

## Fallback:
The app automatically shows placeholder images with the first letter of the skill name if a logo file is missing.
